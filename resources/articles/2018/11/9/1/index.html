So what is this clock?

Well the clock is simply 3 items which one of the is a div that contains the other 2 images. The div has a background of a clock where the 2 other images are the arrows.

Hour arrow is rotated at (Hour / 12)*360 and the 12 is replaced with 60 for the minute arrow. This is to get the percentage of the full cycle and then transform it into the degree. The image then gets their transform class edited with the corresponding degree.

Html file now has "en" set as a language.

Nothing much. Just added lang="en" in the html tag. This is so that other applications can check the langue. For exaple for translation reasons. It may also be used to make sure that non-english users will not get this site but an site corresponding their language. (Which I may plan to add later :D)
